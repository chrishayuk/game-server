# game-server

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run index.tsx
```

This project was created using `bun init` in bun v[5:0]u8@1029b1c56. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.
